Steps to Git Commit:
1.Use the Source Control from the VsCode tool.
2.Clone the repository for the instance.
3.Assign a message on the text field from the Source Control 
4.Then select the files from changes to commit.
5.Click on the + icon from the right of the changed files and Stage them.
6.Once Staged CLick Commit and Sync.
7.Files will be merged.


Packages and softwares needed:
1.NodeJS any version preferred
2.Text editor(VS code / Atom)
3.mongoDB:
    mongod-https://www.mongodb.com/try/download/community
    mongoShell-https://www.mongodb.com/try/download/shell
4.Postman app.
